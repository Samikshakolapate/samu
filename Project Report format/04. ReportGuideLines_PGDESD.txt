Number of pages & Sequence in Project Report Should be as follows:
1. Front Page
2. Certificate
3. Acknowledgment
4. Abstract
5. Index
6. List of Tables & Figures
7. Introduction - 2 Page
8. Literature Survey - 10 Pages
     - Conceptual Study
     - History
9. Project Architecture - 10 Pages
     - Block Diagram
     - Project Design & Explaination
10. Hardware Description - 5 Pages
     - Development machine Desc
     - Development board Desc (if any)
     - Hardware processor, PCI Cards, Peripherals (as applicable)
11. Software Requirements & Specification - 2 Pages
     - Toolchains, IDEs, Used Tools (e.g. minicom, dd, flash magic, ...)
12. Source Code Explaination - 8 Pages
     - Used Kernel APIs, System Calls, Library Functions, Structures, etc.
     - User defined Functions & Structures
     - DO NOT COPY SOURCE CODE - ONLY EXPLAIN 
13. Testing: 3 Pages
     - Test Procedures
     - Screen Shots
14. Future Scope - 1 Page
15. Conclusion - 1 Page
16. References - 1 Page
     - Books - Chapter Nos.
     - Website URLs
